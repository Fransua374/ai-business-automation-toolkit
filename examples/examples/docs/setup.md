# Setup Guide

## Requirements

- Python 3.11+
- Git
- VS Code

## Installation

Clone the repository:

```bash
git clone https://github.com/Fransua374/ai-business-automation-toolkit.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python app.py
```
